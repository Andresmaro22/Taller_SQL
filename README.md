# Taller_SQL
Ejercicios entregables

El Último Salto

El presente taller se desarrolla en el contexto de “El Último Salto”, una tienda especializada en paracaidismo y deportes extremos ubicada en Pasto, Nariño. La base de datos utilizada (tienda.db) contiene cuatro tablas normalizadas con información sintética de más de 1,000 transacciones realizadas en los últimos dos años, permitiendo simular un entorno real de análisis comercial.

La tienda comercializa principalmente paracaídas (militares, deportivos y de emergencia) y accesorios de seguridad como cascos, altímetros, trajes de vuelo y gafas. Además, cuenta con una lógica comercial especial: el cupón “ULTIMO_SUSPIRO”, que otorga un 30% de descuento exclusivo para clientes mayores de 60 años, conocidos como los “saltadores de oro”. Este elemento introduce escenarios de análisis financiero y estratégico dentro del taller.

A lo largo de los ejercicios se aplican consultas SQL que abarcan:
- Consultas básicas con SELECT y WHERE.
- Funciones de agregación como SUM, COUNT, AVG, MAX y MIN.
- Agrupaciones con GROUP BY.
- Uniones entre tablas mediante JOIN.
- Uso de funciones de fecha.
- Subconsultas y CTEs para análisis más avanzados.
- Cálculos financieros derivados del uso de descuentos.

Cada ejercicio fue desarrollado de manera independiente y se encuentra almacenado en archivos separados con extensión .sql. En cada archivo se incluye la solución correspondiente a la pregunta planteada, permitiendo ejecutar, revisar y validar cada consulta de forma autónoma dentro del gestor de base de datos.

Esta organización facilita la comprensión de la lógica aplicada en cada caso y permite analizar el comportamiento comercial de la tienda, identificar patrones de venta, clientes de alto valor y evaluar el impacto financiero del cupón promocional.

En conclusión, este taller no solo resuelve consultas técnicas en SQL, sino que simula un escenario real de análisis de datos aplicado a una tienda especializada, fortaleciendo las habilidades en manejo y análisis de bases de datos relacionales.
